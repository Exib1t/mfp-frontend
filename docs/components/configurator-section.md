# ConfiguratorSection

Homepage teaser for the wigwam configurator feature.

## Purpose

Placed immediately after HeroSection on the homepage. Showcases the configurator as a key differentiator — lets visitors see the customization options at a glance and drives them to /configurator.

## Layout

Two-column: content (left) + visual preview (right).

### Content column
- Overline label
- H2 heading
- Body description
- Feature list: size / fabric / color / personalization
- CTA button → /configurator

### Visual column
- Color swatches grid (static, showing available palette)
- Size badges row
- Label "X+ комбінацій"

## Props

None — static content, all copy is hardcoded.

## CSS block

`configurator-section`

## Variants / modifiers

None.
